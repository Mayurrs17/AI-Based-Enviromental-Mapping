## Troubleshooting

- **Can't see recommendations?** 
  - Make sure you've planned a route
  - Check if you have preferences set
  - Recommendations only show when relevant

- **Can't see offline routes?**
  - Plan at least one route first
  - Routes are saved automatically
  - Click "ğŸ’¾ Offline Routes" to expand

- **Can't see alerts?**
  - Plan a route first
  - Alerts only show when there are environmental hazards
  - Check right sidebar

- **Install button not showing?**
  - Use Chrome/Edge browser
  - Visit site multiple times
  - Button appears automatically when PWA is installable
